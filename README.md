# E7-SecretShopAssistant

WHAT TO IMPLEMENT:
- GET CONNECTION TO THE GAME (EMULATOR OR GOOGLE PLAY GAMES MAYBE?)
- CONNECT THROUGH ADB
- GET INITIAL SCREENSHOT
- DETECT CURRENCIES
- MOVE TO SECRET SHOP IF NOT IN IT ALREADY
- ACCEPT PARAMETERS BEFORE STARTING:
    - GOLD LEFT UNTIL STOP
    - SKYSTONES LEFT UNTIL STOP
- LOOP:
    - DETECT SHOP ITEMS
    - DETECT BUY BUTTONS
    - BUY ITEMS ACCORDING TO PRIORITY:
        - MYSTIC MEDALS
        - COVENANT BOOKMARKS
    - REFRESH SHOP
    - CHECK IF GOLD OR SKYSTONES LEFT

ISSUES:
- DIFFERENT GAME RESOLUTIONS COULD BREAK DETECTION FOR THE SHOP ITEMS.
